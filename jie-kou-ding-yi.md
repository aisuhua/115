## 创建文件夹



	POST /Resourcefolder/create


### Parameters

| Name    | Type      | Description                          |
|---------|-----------|--------------------------------------|
| type			| String			|  <p>素材类型.</p>							|
| parent_id		| String			|  <p>父文件ID.</p>							|
| team_id		| String			|  <p>团队ID.</p>							|
| title			| String			|  <p>文件名.</p>							|

### Success Response

Success-Response:

```
HTTP/1.1 200 OK
{
  "code": "0",
  "msg": "Success",
  "result":"true"
}
```
