## 创建文件



	POST /fileopera/create


### Parameters
 ```
   {
     pid:xxxx,
     path:xxxx
   }
 ```

### Success Response

Success-Response:

```
HTTP/1.1 200 OK
{
  "code": "0",
  "msg": "Success",
  "path":"xxxx",
  "name":'xxxxx',
  "f_id":'xxxxx',
  "error":''
}
```
